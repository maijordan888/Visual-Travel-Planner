# 切換備選方案

## 步驟

1. 在行程時間軸上找到已確認的景點節點。
2. 點擊節點底部的「顯示 N 個備選方案 (可切換)」按鈕展開備選列表。
3. 在備選列表中找到想切換的景點。
4. 點擊「切換採用」按鈕。
5. 原本的景點會自動成為備選方案，新選的景點會成為主要選擇。

## 預期行為
- 切換前：A (confirmed), B & C (options)
- 選擇 B 後：B (confirmed), A & C (options)
- 不會有資料遺失或重複。

## 注意事項
- 「移除」按鈕會永久刪除備選方案，不可復原。
- 可通過「開啟地圖/交由 AI 推薦」添加更多備選方案。
